# git-setup-config
A small script to set your user details in the global configuration.

It will try to infer your name and Eucalyptus email from your laptop
automatically. If these values are incorrect, you can provide alternative
details instead.

## Usage
You can clone and execute, or you can:

```sh
curl -s https://raw.githubusercontent.com/jordanocokoljic-euc/git-setup-config/refs/heads/main/git-setup-config.sh | sh
```
